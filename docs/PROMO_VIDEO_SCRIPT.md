## üé® Visual Style Guide

### Colors:
- **Primary:** `#3B82F6` (Blue)
- **Primary Gradient:** `from-primary/90 to-primary`
- **Success:** `#10B981` (Green)
- **Warning:** `#F59E0B` (Orange)
- **Error:** `#EF4444` (Red)
- **Background:** White (light) / `#0C1222` (dark)
- **Glassmorphism:** `bg-card/80 backdrop-blur-md`

### Typography:
- **Font:** Inter (sans-serif)
- **Headings:** Bold, tracking-tight
- **Body:** Regular, text-base
- **Accent Text:** Gradient text (blue ‚Üí purple)

### Animations:
- **Fade In:** `animate-in duration-500`
- **Hover Lift:** `hover-lift` (translateY -4px)
- **Hover Glow:** `hover-glow` (box-shadow primary)
- **Scale In:** `animate-scale-in`
- **Shimmer:** `shimmer` for loading states

### Icons:
- **Library:** Lucide React
- **Size:** 16px (h-4 w-4) to 24px (h-6 w-6)
- **Colors:** `text-muted-foreground` or brand colors
