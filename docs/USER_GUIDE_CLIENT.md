### Q: Can I cancel my order?

**A:** Yes, if status is still "PENDING_PAYMENT" (before you pay). Contact support after payment to discuss options.

### Q: I didn't receive crypto, what to do?

**A:** Check:
1. Order status - is it "COMPLETED"?

3. Blockchain explorer - search for transaction hash from order details
4. Network - did you check correct blockchain? (e.g., USDT has multiple networks)

**Still not showing?** Contact support with order ID.

### Q: How do I verify transaction on blockchain?

**A:** From your order details:
1. Find "Transaction Hash" (also called TX Hash)
2. Copy the hash
3. Go to blockchain explorer:
   - Bitcoin: blockchain.com or blockchair.com
   - Ethereum: etherscan.io
   - Tron: tronscan.org
   - Solana: explorer.solana.com
4. Paste hash and search
5. See your transaction with confirmations
